# Online Quiz System

## Description
An online quiz platform built with Java, Spring Boot, Hibernate, and MySQL.

## Setup
1. Clone repository
2. Configure `application.properties` for your MySQL database
3. Run: `.\mvnw.cmd spring-boot:run`
4. Open in browser: `http://localhost:8080`

## Tech Stack
- Java, Spring Boot
- Hibernate, JPA
- Thymeleaf
- MySQL
